# 微信元宝 AI · 社交安全带 Demo

> 一个面向投资人/内部评审的高保真可交互 Web Demo。模拟微信聊天场景，展示元宝 AI 如何作为"只对用户可见的社交安全带"，轻量增强用户的聊天、群聊、朋友圈和语音体验。

## 一句话定位

**元宝不是替你聊天，而是帮你更好地聊天。**

## 启动

```bash
npm install
npm run dev
```

打开 [http://localhost:5173](http://localhost:5173) 即可看到。

> 项目使用 React + Vite + TypeScript + TailwindCSS + react-router-dom，无任何后端，AI 输出全部为本地 mock。

## 页面 & 功能

底部 Tab 五个入口：

| 路由 | 内容 |
| ---- | ---- |
| `/` | 首页 / 六大能力总览 |
| `/chat` | 微信聊天列表（含 7 个不同关系类型对话） |
| `/chat/:id` | 聊天对话页（隐形提词器、上下文提醒、潜台词、语音转意图） |
| `/moments` | 朋友圈发布页（素材 + 心情 + 风格 → 文案与隐私守门） |
| `/catchup` | 今日社交补漏（每天 3-5 条值得回的消息） |
| `/pitch` | 投资人速览 / Pitch deck |

## 六大能力 Demo 入口

1. **社交隐形提词器** → `/chat/linxiaoman` 输入 "那你就休息呗" 触发共情提醒；`/chat/boss-zhou` 输入 "哪里有问题" 触发防御语气提醒；`/chat/work-group` 输入 "这个方案不太行" 触发群礼貌提醒；`/chat/echo` 输入 "你每次都这样" 触发吵架冷却。
2. **关系上下文提醒** → 任意聊天打开时顶部出现，可展开 / 收起，可"帮我接上话"。
3. **朋友圈文案** → `/moments` 选素材 + 心情 + 风格，生成 3 条建议，支持"换一批""更像我一点"，并展示隐私提醒。
4. **低压社交补漏** → `/catchup` 4-5 条，可忽略 / 稍后提醒 / 帮我回一句 / 进入聊天。
5. **读懂潜台词** → 在聊天页点击带 `读懂潜台词` 标记的消息（如周总的 "这个方案你再看看"）。
6. **语音转成人话** → 点击妈妈或朋友群里的语音消息旁的"元宝听懂"按钮。

## 设计原则

- 所有 AI 提示**只对用户可见**
- **不替你发**，建议只填到输入框
- **不下判断**：用 "可能"、"更稳妥"
- **默认安静**，关键时刻轻轻提醒
